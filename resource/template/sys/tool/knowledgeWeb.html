JavaScript

1.script标签

如果有大量的JavaScript代码，建议将它放入一个单独的文件进行引入。 

{{"" | html}}

只有最简单的代码才直接写在html中，否则，将其放在一个单独的文件中进来。浏览器会下载单独的js文件，如果在不同的页面引入了相同的js文件，则浏览器会从缓存中获取，而不是再次下载。 这样可以节省流量，并使得页面加载更快。

3.数据类型

3.1 javascript中有8种基本数据类型。

number 整数和浮点数 (Infinity 无穷大,NaN 计算错误)
bigint number 大于9007199254740991的整数,或小于-9007199254740991的数用了bigint表示,在默认加一个n即可，eg: const bitInt = 90071992547409919007199254740991n
string 字符串
boolean 逻辑值 true or false
null 代表未知的
undefined 代表未被赋值的
object 对象就是一组“键值对”(key-value)的集合，是一种无序的复合数据集合。
symbol

3.2 typeof 运算符

3.3 舍入

Math.floor 向下舍入 3.1变成3, -1.1变成-2

Math.ceil 向上舍入 3.1变成4,-1.1变成-1

Math.round 向最近的整数舍入 3.1变成3,3.6变成4,中间值3.5变成4

Math.trunc 移除小数点后的所有内容而没有舍入 3.1变成3,-1.1变成-1

3.4 其他数学函数

Math.random() 返回从0到1的随机数

Math.max(a,b,c...)

Math.min(a,b,c...)

Math.pow9n,power) 返回n的给定(power)次幂 Math.pow(2,10) //2的10次幂 1024
